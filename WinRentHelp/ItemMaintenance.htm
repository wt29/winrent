Item File Maintenance

Before you can create contracts, you will need to add fleet items (items) to the system. Access the item maintenance screen from the Main Menu using FA. Options in this menu are

  • Add - Add items to the file
  • Change - Change item details
  • Owner - Change the owner on a number of items - typically out of lease situations
  • Delete - Delete an item from the file. Not possible if an item is on hire
  • History - Examine and adjust the item's maintenance history

Add

WinRent will prompt you with the next sequential item number. You are free to use this number or subsitute one of your own. The item number does not need to be purely numeric. For example "XYZ123" is valid. Winrent will check to see if the number entered is unique. You will be prompted to re-enter if it is not.

An item owner is then requested. The default code, set up in Utility/Details is offered. Enter a valid owner code. You are then presented with the item addition dialog. Fields presented are

  • Model Number - this field is searchable
  • Description - Description of the item - not searchable
  • Serial No - Seachable field
  • Product Code - This field allows you to categorise your stock for exampe 7kg Washing machines. See "Entering System Codes" for details on how to create new codes.
  • Status - this is a coded field - There are 2 default status codes "O" = Onhand and "H" Hired. Winrent creates these automatically. You are free to add your own codes e.g. "R" for Repair for stock under repair. See "Entering System Codes" for details on how to create new codes.
  • History fields - If you are entering an existing fleet item and you have history details, you can enter these here. Note that WinRent will update these fields as the item is hired and returned.
  • Rental Amounts - Enter the Monthly, Fortnightly, Weekly and Daily rentals amounts here. These are used during contract creation. They can be overridden at contract creation. You must enter a value for each field. If you don't perform, for example, daily rentals simply enter a value of $1. There is no impact of entering values for rental periods you don't support.
  • Leasing Details - Entering the leasing details (if any) will permit WinRent to calculate Rule 72 leasing payout details. There is a report for this function. Note that this also requires the End of month function to be run.

Hit PgDn to complete and save the item details. As before Esc will abandon (not save) the data entered

Change

Enter the item code to change details including history fields

Owner

This function allows you to change a range of items from one owner to another. Typically these will be items coming "off lease". The function requires entering an "old" owner and a "new" owner. Once entered you can enter a number of item codes. You will be prompted to confirm each change

Delete

Delete old stock items from the system using this function. Items that are on hire cannot be deleted. All item history is also deleted.

History

Used to add repair history to items
