Contract Enquiry

Contract enquiries are by far the most common lookup in Winrent. WinRent presents a flexible Contract enquiry method. All contract enquiries present the same screen

  • Contract Number - if your enquiry text begins with a number Winrent will assume you are looking up by contract number.
  • Surname - If the enquiry text begins with a letter from A-Z Winrent will lookup using the surname
  • Deposit Book Number - prepending your enquiry number with a '*' will lookup using the deposit book number
  • Address - prepending using the "/" will search the first line of the address. i.e. if you put /123 then WinRent will show all addresses with "123" in the start of the text
  • Suburb - using a "+" at the start will force WinRent to search by Suburb i.e. +Manly will show all accounts in Manly
  • Phone - a "-" will allow a search by phone number

For all enquiries other than contract and deposit number, if there is more than one match, Winrent will display a pick list as follows

Navigate up and down the list using Page Up (PgUp) and Page Down (PgDn) for a full page. The keyboard arrow keys will move between individual contracts. Hit to select the required entry. Note also that the left and right arrows will also display further columns beyond the right hand side of the dialog
A typical contract is displayed

The screen shows the first hirer on the contract on the left side of the screen. If the contract has more than one hirer, you can select the others by using the right (and left) arrow keys. The purple section on the right shows contract details including the paid to date, the installment amount and the bond details. The panel at lower right shows the items on the contract. Note that only the first 3 items are shown.

There are several extra functions available here. Hitting "C" will allow the "adhoc" comments to be edited. Just add/change the details and hit enter. The "long" comments can be accessed by hitting F4. These comments are effectively "unlimited" and are stored in a text file (.txt) in the "mcomments" folder under the WinRent folder. These comments are stored by contract number i.e. the file 10001.txt is the comment folder for contract 10001. They can be edited, if required using an external text editor however WinRent cannot handle text from non "text editor" programs such as Microsoft Word.

A number of other functions are also present. They are activated by the following function keys

  • F5 - Shows the full item details on the contract
  • F6 - Shows the transactions on the contract
  • F7 - Displays the arrears history
  • F11 - Change the "Status" field

F5 - Items on contract

Shows the items on the contract. scroll up and down to select and hit enter to see the full details of each item

F6 -Transactions

Shows all transactions on the selected contract, sorted latest first. Hit the right arrow to scroll sideways if required. You can change the comments on any transaction by hitting

F7 - Arrears

Shows the arrears history (how often the contract went into arrears)

F11 - Status

Allows you to change the "Status" flag on the contract
