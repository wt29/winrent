Utilities Details

This screen sets up a number of Winrent System variables. Most of these variables are system wide, the printer variables are on a "per workstation" basis. This meanes that the printers need to be set up on each workstation.

Address Lines

Winrent will use these values as output on reports as required

Default Period of Grace

Winrent will add the value here (if any) to the days a user is in arrearrs before sending out a letter. 

Month for end of year

Used in asset value calculations. If your end of financial year is June then insert "06" into this field

Default Owner

Provides a default owner value when entering new stock

Item Start number

As you add items to  WInrent, it adds a sequential number (the Item ID) to each record. This field allows you to select the starting value. You can change this at will

Contract Start Number

Each contract is identified by a sequential number. This field allows you set the starting value. In a  new WinRent installation, this will start from 1. Note that the value of "0" (Zero) is reserved by WinrRent and cannot be used.

Printers

Printers are set on a "per workstation" basis, that is you need bo set the printers on each workstation individual. There are 2 printers supported, a "Report" and a "Letter" printer. The majority of output is directed to the report printer. If you only use one printer for both then enter the same details in both fields.

WinRent supports both local and network printers. The most important thing to remember is the the printer name is case sensiitive and uses the "Windows Printer Name" for example, if you have a local printer called "Epson" then you must enter "Epson" in the printer string with the "E" as a capitalised. Shared printers on other machines are usually entered as a "UNC" string. e.g you have a printer Epson on a workstation "Computer1" - the string "\\Computer1\Epson" would be entered.

Late Fee Details

Sets up  the abilty to charge late fees and the amounts (fi any) on the arrears letters

Editor Program

Winrent can output most reports to the screen if required. To do this, it ouputs the report to a text file and then calls an editor program to display it. If you don't have a preferred editor, enter "notepad.exe" otherwise enter the full path to your preferred text editor.
